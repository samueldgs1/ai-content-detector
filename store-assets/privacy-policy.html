Privacy Policy

Last updated: June 2026

Sift is a Chrome extension that detects AI-generated content on webpages. This policy explains what data the extension accesses and how it is handled.

Data We Collect

Sift does not collect any personal data. The extension does not have a server, does not create accounts, and does not transmit any information about you or your browsing to any third party.

What the Extension Accesses

To function, Sift reads the text content and image URLs of pages you visit. This processing happens entirely inside your browser. The content is never stored beyond your current session, and never sent anywhere.

  • Page text — read locally to score for AI patterns. Never transmitted.
  • Image URLs — checked against a local list of known AI platform domains. Never transmitted.
  • Settings — stored locally using Chrome's built-in sync storage (your whitelist, sensitivity preference, mode). Synced by Chrome across your own devices if you have sync enabled.

Optional API Key

If you choose to enter an API key in Settings, it is stored locally in Chrome's storage and sent only to the API endpoint you configured (e.g. api.anthropic.com) when deeper analysis is requested. Sift never sees or stores the results on any server.

No Analytics or Tracking

Sift contains no analytics, no crash reporting, no telemetry, and no advertising. There is no way for the developer to know how many people use the extension or what sites they visit.

Changes

If this policy changes in a meaningful way, the extension version will be updated and the date above will reflect it.
